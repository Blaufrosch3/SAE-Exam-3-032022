Member Enumeration Documentation

◆ LevelState

enum Level.LevelState

Possible States of the currentLevel.

The GameManager decides when to end the level (Win/Loose) upon the current state of the level.

Enumerator
NoElementsLeft 

The Level is "empty".

All Elements of the ElementGrid were removed by the player through Click-Events.

NoMoreMovesPossible 

There are still Elements in the level but the player can't perform moves.

Means that there are no Elements with Neighbours of the same type (Element.ElementType) left.

FurtherMovesPossible 

This is the DEFAULT state.

This State remains as long as the player has the possibility to remove further Elements from the level.

Checks the state of the current Level.
